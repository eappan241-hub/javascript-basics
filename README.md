# Student Grade Calculator

## Theory

This JavaScript program calculates a student's **grade**, **pass/fail status**, and **remark** based on the marks obtained.

## Features

- Stores marks and bonus marks using variables  
- Uses the assignment operator (`+=`) to add bonus marks  
- Uses `if-else-if` statements to determine the grade  
- Uses a ternary operator (`? :`) to determine Pass or Fail  
- Uses a `switch` statement to assign remarks  
- Displays the output using `console.log()`  

## Grade Criteria

| Marks Range | Grade |
|------------|-------|
| 90 - 100   | A     |
| 70 - 89    | B     |
| 50 - 69    | C     |
| Below 50   | F     |

## Remarks

| Grade | Remark            |
|------|------------------|
| A    | Excellent!        |
| B    | Good              |
| C    | Average           |
| F    | Needs Improvement |
